# MTN Nigeria Homepage Clone

This project is a recreation of the MTN Nigeria homepage built using only HTML and CSS.

## 🚀 Project Overview

As part of my learning journey, I decided to move beyond tutorials and start rebuilding real-world websites to better understand how production-level interfaces are structured.

This project focuses on translating a real design into code while paying attention to layout, spacing, and responsiveness.

## 🎯 Objectives

- Practice semantic HTML structure
- Improve CSS layout skills (Flexbox & Grid)
- Understand real-world UI patterns
- Build responsiveness across screen sizes

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox & Grid)

## 📚 What I Learned

- Structuring complex layouts using semantic HTML
- Managing spacing, alignment, and positioning
- Breaking down a real-world design into smaller components
- Improving attention to detail in frontend development

## ⚠️ Challenges Faced

- Replicating the navigation bar layout
- Handling spacing consistency across sections
- Making the layout responsive for different screen sizes

## 🔄 Future Improvements

- Add JavaScript for interactivity
- Improve animations and transitions
- Optimize for performance
- Enhance mobile responsiveness
